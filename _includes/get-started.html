Getting started

There are three options of FIND depending on how you would like to use it: "Personal," "Home," and "Business." Each of these options has their own specific set of features.

1. Personal - Free

By far, this is the simplest and easiest way to start. Though you won't have a complete set of features using this option, this is perfect if you just want to try it out.

You can download all the source code to build your own server, app, command line programs, if you want. We have kept the useful part of FIND open-source, so that you can use it for your own projects. This option also only has a limited set of features, however you can run it at home on whatever system you want.
